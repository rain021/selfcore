SelfCore Chrome Extension

MirrorSync v2 — AI Context Bridge

Features

Auto Context Injection

Automatically inject your SelfCore profile when chatting with AI

Multi-Platform Support

Works with Claude, ChatGPT, Gemini, and Mistral

Keyboard Shortcut

Press Ctrl+Shift+Space to inject context anytime

Injection Logging

Track every injection in the SelfCore Insights dashboard

Prerequisites

Make sure SelfCore backend (selfcore.py) is running on localhost:8100 before using the extension.

Installation Steps

1

Open Chrome Extensions Page

Open Google Chrome and navigate to:

chrome://extensions

2

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the page.

3

Load Unpacked Extension

Click "Load unpacked" button, then select the extension/ folder inside your SelfCore directory:

C:\Users\user\Desktop\SelfCore\extension

4

Pin the Extension

Click the puzzle icon in Chrome toolbar, then click the pin icon next to "SelfCore MirrorSync" to keep it visible.

5

Verify Connection

Click the SelfCore icon in the toolbar. If the status shows a green dot with "Connected", you're all set!

How to Use

A

Manual Injection

On any supported AI site, press Ctrl+Shift+Space

Your SelfCore context will be prepended to the input field.

B

Auto-Inject Mode

Open the SelfCore popup and enable "Auto-inject on send". Context will be automatically injected every time you press Enter to send a message.

Troubleshooting

"Not running" status: Make sure selfcore.py is running. Start it with: py -3 selfcore.py
Extension not injecting: Reload the AI site tab after installing the extension. Content scripts only load on fresh page loads.
Badge not showing: Check that the extension has permission for the current site in chrome://extensions.
